1. Before running the code, the format of image slices and VOI files should be transfer into ".nii.gz" format.

2. The name of the folder was the patients ID, and the folder only contain the "img.nii.gz" file and "roi.nii.gz".

3. Then, run the "main" code in the Command Window of the Matlab software in the laptop computer. 

4. The program will display pop-up window, just need to select one patient folder which only contain the "img.nii.gz" file and "roi.nii.gz".

5. After program running completed, one patient should have five excel files as the result datas.
